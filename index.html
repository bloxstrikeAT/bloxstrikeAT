Shortcuts

✕
Key / ActionWhat it does
SSave every unsaved slot at once.
1-9 1-4 or h/d/w/m 1-4Toggle a mission — slot → category → mission number. Category can be typed as a digit (1-4) or a letter (h/d/w/m = hourly/daily/weekly/monthly) — both work. Skip the slot digit if there's only one slot. Hover a mission's square to see its code (e.g. 1d1).
1-9 1-4 or h/d/w/m RReset a whole mission category — slot → category → R. Skip the slot digit if there's only one slot.
1-9 RReset every mission in that slot at once — slot → R (skip the category). With only one slot, plain R does this.
1-9 SSave just one slot — slot → S. (With only one slot, plain S already does this.)
M 1-9 h/d/w/m 1-4Jump straight to editing a mission's name — M → slot → category → mission number. Switches the mission tracker card to that category and puts your cursor right in the name field, text selected and ready to type over. Skip the slot digit if there's only one slot.
Shift ,Undo the last action.
Shift .Redo the last undone action.
EscCancel an in-progress keyboard sequence, or close this panel.
Scroll over any number field±1 per notch, hold Shift for ±5, hold Ctrl for ±10.
Scroll over the mission timerEdits whichever segment your cursor is over — days, hours, minutes, or seconds.
⋮⋮ handleDrag a row to reorder slots.
↺ resetClears that stat/category back to zero.
